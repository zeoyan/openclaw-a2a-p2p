# Publishing checklist

Before pushing this repository to GitHub, verify:

- README matches the current code
- examples use the correct plugin id: `a2a-p2p`
- docs mention dedicated routing sessions
- docs mention route auth requirement (`plugin` or `gateway`)
- bearer token placeholders are still placeholders
- no local secrets or tokens are committed
- no local machine-specific paths remain except generic examples
- install and verify scripts are present and executable
